# Galeria de Fotos San Valentin

## 1) Descripción del Proyecto

Pequeña galería de fotos con una estética romántica para celebrar el Día de San Valentín. La página muestra un título con efecto de brillo y una galería animada de seis imágenes que se superponen y se desplazan con un carrusel CSS puro.

## 2) Tecnologías Utilizadas

- HTML5
- CSS3 (incluye variables, keyframes y media queries)
- Fuente: Comic Neue (Google Fonts)

## 3) Conceptos de CSS Aplicados: Propiedades, Funciones y Animaciones

- Propiedades: Uso de propiedades como `display: grid`, `z-index`, `box-shadow` y `border` para controlar el diseño, el apilamiento y el aspecto visual de las imágenes en la galería.

- Funciones: Empleo de variables CSS (`--d`) y funciones como `calc()` y `var()` para calcular retardos y tamaños dinámicos, facilitando control sobre animaciones y dimensiones sin JS.

- Animaciones: Definición de `@keyframes` (`slide`, `z-order`) y `animation-delay` para crear el efecto de carrusel; las animaciones manejan traslaciones, rotaciones y cambios de orden de apilamiento.
