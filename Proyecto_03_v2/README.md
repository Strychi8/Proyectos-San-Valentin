# Decision Si/No (v2)

## 1) Descripcion del Proyecto

Este proyecto es una tarjeta/interacción web de San Valentín que presenta un fondo de video, GIFs animados y dos botones "Si" y "No" para generar una secuencia de respuestas visuales. Está diseñado como una experiencia sencilla y romántica que cambia imágenes, texto y estilos según la interacción del usuario.

## 2) Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (DOM, eventos y manipulación de estilos)
- Recursos multimedia: GIFs, video WebM e imágenes

## 3) Conceptos de CSS y JavaScript aplicados

- CSS: Uso de `position: fixed` y `object-fit: cover` para un fondo de video responsivo; propiedades de layout flexbox para centrar contenido y controles de tamaño/efectos (`box-shadow`, `transition`, `scale`) para interacción visual.
- JavaScript: Manejo del DOM y eventos (`addEventListener`) para responder clicks; cambio dinámico de atributos (`src`, `innerHTML`) y estilos con `Object.assign` para animar la interfaz y crear una secuencia de pasos.
