# Carta Interactiva para San Valentin

## 1) Descripcion del Proyecto

Una tarjeta interactiva de San Valentín que simula un sobre con una carta en su interior. El proyecto muestra un mensaje romántico y presenta elementos visuales y animaciones que recrean un sobre con solapas y un corazón animado.

## 2) Tecnologias Utilizadas

- HTML5 
- CSS3
- JavaScript
- Fuente: Marck Script (Google Fonts)

## 3) Conceptos CSS y JavaScript Aplicados

CSS: Se usan posicionamiento y transformaciones para construir el sobre y las solapas, además de transiciones y animaciones (keyframes) para el corazón y las aperturas; el diseño incluye técnicas responsivas para mantener la presentación en distintos anchos.

JavaScript: Maneja la interactividad mediante listeners y manipulación del DOM, alternando clases para disparar animaciones y cambios de estado (abrir/cerrar sobre); también gestiona sincronización y temporización de efectos para mejorar la experiencia.
